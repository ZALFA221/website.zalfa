about me

Perkenalkan nama Saya zalfa Azhar Ashidiq, teman-teman saya biasanya memanggil saya Zull. Saya tinggal di kampung sekartoya,desa pasirhuni,kecamatan cimaung,kabupaten bandung, kalian pasti tau dengan tempat wisata gunung puntang nah rumah saya tidak jauh dari sana jarak nya. Saya mudah bergaul dan menyesuaikan dengan lingkungan sekitar serta saya tergolong ke dalam orang yang humoris. Jadi dengan perkenalan ini saya harap bisa berteman baik dengan kalian semua 😁
